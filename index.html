Display Specification

Model: SH1106

Size: 1.3 Inch

Resolution: 128x64

Interface: I2C

Pin Definitions

Component GPIO
OLED SDA GPIO 4
OLED SCL GPIO 5
Control Button GPIO 3
VCC 3.3V
GND GND

How To Build

  1. Connect SH1106 SDA → GPIO 4
  2. Connect SH1106 SCL → GPIO 5
  3. Connect Button → GPIO 3
  4. Connect 3.3V & GND
  5. Power via USB
  6. Click Install Firmware
  7. Done
